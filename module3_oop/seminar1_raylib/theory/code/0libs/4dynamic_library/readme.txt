Подключение динамической библиотеки библиотеки

Чтобы скомпилировать эту программу, нужно подключить к файлу main.cpp динамическую библиотеку из папки cube.
Для этого нужно использовать опции -I, -L и -l компилятора g++.

    g++ main.cpp -I./cube/include -L./cube/lib -lcube



Запуск программы на Windows(MSYS2 MinGW):
Нужно, чтобы было выполнено одно из следующих:
1) Динамическая библиотека cube.dll должна лежать в той же папки, что и исполняемый файл.
2a) Переменная среды Path должна содержать путь до динамической библиотеки.
2b) Можно временно изменить Path при запуске программы:

     PATH=./cube/lib:$PATH ./a.exe
